MdcButton

Selector: button[mdc-button] a[mdc-button]

Exported as: mdcButton

Properties

Name Description
primary: boolean Colors the button with the primary theme color.
secondary: boolean Colors the button with the secondary theme color.
dense: boolean Compresses the button text to make it slightly smaller.
raised: boolean A contained button that is elevated upon the surface.
unelevated: boolean A contained button that is flush with the surface.
outlined: boolean A contained button that is flush with the surface and has a visible border.
disabled: boolean Disables the button.
label: string Indicates the element containing the button's text label.

Methods

focus() Focuses the button.
