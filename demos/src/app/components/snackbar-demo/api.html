Service

MdcSnackbar

Service to open MDC snackbars.

Properties

Name Description
afterDismiss: Observable 
MdcSnackbarDismissReason
 Gets an observable that is notified when the snackbar is finished closing.

Methods

open(message: string) Open snackbar with message.
open(message: string, action: string) Opens snackbar message and action.
open(message: string, action: string, config?: MdcSnackbarConfig) Opens the snackbar with optional configuration.

MdcSnackbarConfig

timeoutMs?: number Value must be between 4000 and 10000 or an error will be thrown. Defaults to 5000 (5 seconds).
stacked?: boolean Positions the action button/icon below the label instead of alongside it. Defaults to false.
leading?: boolean Positions the snackbar on the leading edge of the screen.
trailing?: boolean Positions the snackbar on the trailing (right) edge of the screen.
direction?: string The layout direction of the snackbar content. Default is 'ltr'.
dismiss?: boolean Show dismiss ("X") icon. Default is false.
closeOnEscape?: boolean Whether the snackbar closes when it is focused and the user presses the ESC key. Default is true.
classes?: string | string[] Add a CSS class or an array of classes.
actionClasses?: string | string[] Add a CSS class or an array of classes to the action button.
dismissClasses?: string | string[] Add a CSS class or an array of classes to the action icon.
