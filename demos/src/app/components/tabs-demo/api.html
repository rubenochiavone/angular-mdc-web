MdcTabBar

Selector: mdc-tab-bar

Exported as: mdcTabBar

Properties

Name Description
stacked: boolean Indicates that the tab icon and label should flow vertically instead of horizontally.
fixed: boolean Indicates that the tab should shrink in size to be as narrow as possible without causing text to wrap.
fade: boolean Sets up the tab indicator to fade in on activation and fade out on deactivation. Default is false.
activeTabIndex: number Set the active tab by index. (Default is 0). Set to -1 for no initially selected tab.
iconIndicator: string A material icon to display in the center of the tab.
useAutomaticActivation: boolean If true, permits using movement keys to switch tabs.
focusOnActivate: boolean Sets whether the tab should focus itself when activated. Defaults to true.
align: string Sets the elements inside the scroll content element to be aligned to the start, center or end of the scroll content element.

Valid values: 'start' | 'center' | 'end' * Only for Fixed Tabs.

Methods

activateTab(index: number) Activates the Tab at the given index.
scrollIntoView(index: number) Scrolls the Tab at the given index into view.
getActiveTabIndex(): number Returns the index of the active Tab.
getActiveTab(): MdcTab | undefined Returns the MdcTab that is active.
getTabIndex(tab: MdcTab): number Returns the MdcTab index for given tab.
disableTab(index: number, disabled: boolean) Control whether or not the tab is in a disable state.

Events

activated: (source: MdcTabBar, index: number, tab: MdcTab) Emitted when a tab is activated.

MdcTabBarScroller

Selector: mdc-tab-scroller

Exported as: mdcTabBarScroller

Properties

Name Description
scrollTo(scrollX: number) Scrolls to the scrollX value.
incrementScroll(scrollXIncrement: number) Increments the current scroll value by the scrollX value.
getScrollPosition(): number Returns the current visual scroll position.

MdcTab

Selector: mdc-tab

Exported as: mdcTab

Properties

Name Description
id: string Unique id of the tab (auto generated if not supplied).
label: string Optional. Adds a label in the tab.
icon: string Optional. Indicates a leading icon in the tab.
disabled: boolean Optional. Control the disabled state of the tab.
stacked: boolean Indicates that the tab icon and label should flow vertically instead of horizontally.
fixed: boolean Indicates that the tab should shrink in size to be as narrow as possible without causing text to wrap.
focusOnActivate: boolean Sets whether the tab should focus itself when activated. Defaults to true.

Methods

getTabBarParent(): MdcTabBarParentComponent Interface for communicating with ancestor MdcTabBar.
focus(): void Sets focus to the tab.

Events

interacted: (detail: MdcTab) Emitted when a tab is interacted with.
